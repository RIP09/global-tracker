# Global Tracker - Real-Time Location Platform

This is a **legal, consent-based, real-time location tracker** built with **Python Flask, Firebase, and Leaflet/OpenStreetMap**.  
It works on **desktop browsers** and can be embedded in **Android apps** via WebView or Flutter.

---

## Features

- 🌍 Real-time moving markers for all users
- 🔐 Firebase Authentication (Email/Password)
- 👥 Multi-user dashboard
- 💻 CMD + Browser interface
- ⚡ Fully free (no Google Maps API key required)
- 📡 Works worldwide with Firebase Realtime Database
